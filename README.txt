Moleculix ThirdParties 
============================================
Moleculix is a library for molecular visualization and generation of molecular geometric related information. 
Moleculix contain, in particular, functionality to generate NMR related tensor information from molecular geometry.
Tensor data can be used as input to NMR simulation engines.

Please see www.moleculix.com for info, updates and the latest code.
